# VacuumCleaner
This is a browser web app that solves simple *AI* problem that is known as __"vacuum cleaner"__. Designed with pure html, css and js.
## Demo
Demo of this web app you can find at [Vacuum Cleaner](https://yaroslavgrushko.github.io/VacuumCleaner/).
## Getting Started
To run this website you only need to download it soure code and launch index.html in a browser. (It was tested on Google Chrome, Opera and Firefox).
